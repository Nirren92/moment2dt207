About

Denna sida tillåter användaren att lägga till samt ta bort arbetserfarenheter.

Hemsidan bygger på API anrop till en webservice som hostas av Render. Även här ligger en Databas i form av Mongodb. information gällande API finnes i readme filen i github.

Riktigt kul att få göra både front och backend. Nu börjar helheten komma på plats och möjligheten att bygga en komplett hemsidan från "scratch" om man vill
